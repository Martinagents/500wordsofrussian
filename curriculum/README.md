# Curriculum

Generated by `curriculum/scripts/generate_deck.py`; validated by `curriculum/scripts/validate_deck.py`.

## Sources

- `wordfreq-ru-2026`: intended independent Zipf-frequency signal from Python `wordfreq`; generator preserves source rank/Zipf-like fields in provenance.
- `rnc-frequency-dictionary`: Russian National Corpus frequency dictionary metadata used as corpus-rank signal. Retrieval should be refreshed from the official RNC frequency dictionary when available.
- `elementary-lexical-minimum`: manually checked elementary/A1 lexical-minimum layer; current MVP records whether items are in the A1-derived coverage band.
- `editorial-communicative-layer`: transparent layer for predicates, repair, family/household relevance, Berlin/travel, and reusable frames.

Retrieval date: 2026-06-17. The checked-in deck is usable without remote downloads. Reproducibility limitation: this MVP includes compact transformed source metadata, not full third-party source dumps; it does not claim native-speaker certification.

## Scoring

The script stores visible components close to `0.40 frequency + 0.20 corpus agreement + 0.20 A1 + 0.20 communicative utility`, plus bonuses/penalties and editorial override reasons.
